1. What is the difference between .map(), filter(), forEach() and find() ?
Ans:

  • .map()
    This method returns a new array with the same length of the original array, but each elements will be transformed based on the condition and the original array will remain unchanged

  • .filter()
    This method returns a new array with only elements that passed the test based on the condition and the original array will remain unchanged

  • .find()
    This method returns only the first element that passed the test based on the condition, if no element pass the test it will return `undefined`

  • .forEach()
    This method iterates over each element of an array and executes a function on it.It does not return anything and does not modify the original array.

2. What is localStorage, sectionStorage and cookies ?
Ans:

localStorage and sessionStorage are methods of storing data in a web browser using JavaScript, while cookies are small text files that are stored on a user's computer by a website. Each of these methods has different use cases and limitations, and the choice of which to use depends on the specific needs of the application.

3. What is event loop and how does it work ?
Ans:

The event loop is a mechanism in JavaScript that allow the code to run asynchronously without blocking the main thread. It continuously checks for new tasks, it executes the task and moves to the next one. This method allows JavaScript to handle user input and other time-consuming operations without slowing down the application.

quizTimer()

This function is like a timer for a quiz game. When you start the quiz, it shows the time left for the quiz to end. Every second, the timer updates the time left on the screen. It also changes the color of the time left every 60 seconds to make the ui more user friendly.

This function is like a quiz. This function execute everytime you answer a quiz. This function checks if you already answered the question before. If you did, it updates your answer. If you didn't, it adds your answer to the list of answers and show all the ans you gave.
